# @daddia/create-space

Scaffold a new daddia delivery workspace.

## Quick start

```bash
cd packages/create-space && pnpm build && npm publish --access public
npx @daddia/create-space my-project --yes --key ACME --use-pnpm
```

### Interactive

```bash
# npm
npm create @daddia/space
npm create @daddia/space my-project
npx @daddia/create-space my-project

# pnpm
pnpm create @daddia/space
pnpm create @daddia/space my-project
pnpm dlx @daddia/create-space my-project

# yarn
yarn create @daddia/space
yarn create @daddia/space my-project

# bun
bunx @daddia/create-space
bunx @daddia/create-space my-project
```
