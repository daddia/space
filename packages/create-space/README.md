# @tpw/create-space

Scaffold a new Crew delivery workspace for Horizon.

## Quick start

```bash
cd packages/create-space && pnpm build && npm publish --access public
npx @tpw/create-space my-project --yes --key ACME --use-pnpm
```

### Interactive

```bash
# npm
npm create @tpw/space
npm create @tpw/space my-project
npx @tpw/create-space my-project

# pnpm
pnpm create @tpw/space
pnpm create @tpw/space my-project
pnpm dlx @tpw/create-space my-project

# yarn
yarn create @tpw/space
yarn create @tpw/space my-project

# bun
bunx @tpw/create-space
bunx @tpw/create-space my-project
```
